# Jacob Aragao - Portfolio

A personal portfolio site built to showcase my projects and work experience. This project was also an opportunity to push my design thinking, applying lessons learned during my internships to create something polished and intentional.

## Tech Stack

- **React 19** + **TypeScript** - Modern component architecture with type safety
- **Vite** - Fast development server with HMR
- **Tailwind CSS v4** - Utility-first styling with custom theme
- **Framer Motion** - Smooth animations and page transitions
- **React Router** - Client-side navigation
